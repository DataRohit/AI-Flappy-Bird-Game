# **AI-Flappy-Bird-Game**

#### **1. Create Virtual Environment**
```bash
$ python -m venv venv
```

#### **2. Activate Virtual Environment**
```bash
$ venv\Scripts\activate
```

#### **3. Install Required Packages**
```bash
$ pip install -r requirements.txt
```

### **4. Run the game.py file**
